# DHPE Content Automation

מערכת אוטומציית תוכן לאתר [dhpe.co.il](https://dhpe.co.il) — דניאל חסיד פרויקטים בחינוך.

## מה המערכת עושה

מפיקה 2 פוסטי SEO בשבוע לאתר הוורדפרס, כולל:
- מחקר מילות מפתח
- כתיבת פוסט 1500-2000 מילים
- חבילת SEO מלאה (Meta, OG, Schema FAQ)
- העלאה אוטומטית כטיוטה לוורדפרס
- מייל הזנה אישי לדניאל לפני כל פוסט
- מייל התראה כשהטיוטה מוכנה

## לוח זמנים

| יום | שעה | פעולה |

| שבת | 17:00 | מייל הזנה לפוסט ראשון |

| רביעי | 08:00 | כתיבה + העלאה לוורדפרס + מייל התראה |

## מבנה התיקיות

```

├── SKILL.md                    # הוראות הפעלה לסקיל

│   ├── post-template.md        # מבנה אחיד לכל פוסט

│   └── schema-faq.json         # תבנית FAQ Schema
├── content-plan/
│   └── 24-posts-plan.md        # 24 נושאים ל-3 חודשים + סטטוס
├── email-templates/
│   ├── feeding-email.md        # תבנית מייל הזנה
│   └── draft-ready-email.md    # תבנית מייל התראה
└── knowledge/
    ├── business-context.md     # הקשר עסקי, טון, הוראות כתיבה
    ├── pricing.md              # מחירון רשמי
```

## Connectors נדרשים

- **WordPress** — לבדיקת פוסטים קיימים ולהעלאת טיוטות
- **Gmail** — לשליחת מיילי הזנה ולקבלת תשובות דניאל

## הרצה ראשונה

1. בדוק את `content-plan/24-posts-plan.md` — ודא שהנושאים נכונים
2. הרץ ידנית מייל הזנה ראשון — בדוק שמגיע תקין
3. הרץ ידנית כתיבת פוסט ראשון — עבור על הטיוטה בוורדפרס
4. רק לאחר אישור — הפעל את לוח הזמנים האוטומטי
